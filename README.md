# Rain-sensor-system-with-LED-and-sound-feedback-Arduino-

## Overview
A simple Arduino-based interactive system using a rain sensor.

## How it works
- When water is detected:
  - LED gradually lights up
  - A short melody is played through a buzzer
- When dry:
  - LED fades out
  - Sound stops
    
## Hardware Used
- Arduino
- Rain sensor
- LED
- Buzzer

## Notes
This project explores basic sensor-based interaction and real-time feedback.
It can be extended into larger interactive installations.
